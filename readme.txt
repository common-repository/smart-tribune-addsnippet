A plugin that make even easier the integration of the <a href="http://www.smart-tribune.com/">Smart Tribune</a> User Feedback application.


== Description ==

A module that allows admins to easily insert Smart Tribune snippet into their website/application.
It is used to inject Smart Tribune code snippet just before body end tag which is the recommended placement for Smart Tribune snippet.

Smart Tribune is a brand new SaaS software that let you personalize your online customer relationship. This module will help Website admin to integrate as easily as possible snippet into their website. 

To learn more about Smart Tribune, feel free to visit our website [Smart Tribune](http://www.smart-tribune.com/)

== Installation ==

1. Extract all files from the ZIP archive, making sure to keep the file structure intact.
2. Upload the `smarttribune-addsnippet` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Go to the `Smart Tribune Plugin` menu which is located under the `Settings` menu in the admin interface.
5. Paste your Smart Tribune snippet to activate the plugin.
